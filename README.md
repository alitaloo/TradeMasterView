## 🎯 功能特性

### 1. 仪表板 (Dashboard)
- 📊 今日信号概览
- 📈 策略表现摘要
- 💹 投资组合分布
- 🔔 重要提醒

### 2. 信号面板 (Signals)
- 📋 实时信号列表
- 🔍 信号筛选/搜索
- 📊 信号统计图表
- 💾 信号历史

### 3. 回测结果 (Backtests)
- 📊 回测结果对比
- 📈 权益曲线图表
- 📉 回撤分析
- 🏆 策略排名

### 4. 策略管理 (Strategies)
- 📋 策略列表
- 📊 策略参数配置
- 🔧 策略启用/禁用
- 📈 策略表现追踪

### 5. 投资组合 (Portfolio)
- 💼 持仓分布
- 📊 收益分析
- ⚠️ 风险指标
- 📈 资产配置

## 🛠️ 技术栈

- **框架**: Vue.js 3 + Vite
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **图表**: Chart.js / ECharts
- **UI 组件**: 自定义组件
- **API 通信**: Axios
- **样式**: CSS3 + Flexbox + Grid

## 🚀 快速开始

```bash
# 进入项目目录
cd TradeMasterView

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📡 API 对接

项目将对接以下数据源：

1. **TradeMaster API** (`http://localhost:8000`)
   - `/api/signals` - 获取信号列表
   - `/api/backtests` - 获取回测结果
   - `/api/strategies` - 获取策略列表
   - `/api/portfolio` - 获取投资组合

2. **本地数据文件**
   - `../TradeMaster/data/backtest_results/` - 回测结果 CSV
   - `../TradeMaster/signals/` - 信号记录
